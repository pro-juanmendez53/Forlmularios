Lección 9: Agrupación de Elementos (<fieldset> y <legend>)

Organizar Visualmente Secciones del Formulario

Cuando un formulario es largo y complejo, puede ser útil agrupar campos relacionados en secciones. Para esto, HTML nos proporciona dos etiquetas semánticas: <fieldset> y <legend>.

  • <fieldset>: Se usa para agrupar un conjunto de campos relacionados en un formulario. El navegador generalmente dibuja un borde alrededor de estos elementos para indicar visualmente la agrupación.
  • <legend>: Se usa para proporcionar un título o una leyenda para el <fieldset>. Debe ser el primer elemento hijo del <fieldset>.

Mejora de la Accesibilidad y Estructura

Más allá del beneficio visual, usar <fieldset> y <legend> mejora enormemente la accesibilidad. Los lectores de pantalla anuncian la leyenda (<legend>) a medida que el usuario navega por los campos dentro del <fieldset>.

Por ejemplo, si tienes un grupo de radio buttons para "Tipo de envío", el lector de pantalla podría anunciar "Tipo de envío: Envío estándar" o "Tipo de envío: Envío exprés", proporcionando un contexto crucial que de otro modo se perdería.

Ejemplo de Código

En este ejemplo, agrupamos la información personal y las preferencias de contacto en dos secciones claras.


<form action="#">
  <fieldset>
    <legend>Datos Personales</legend>
    <div>
      <label for="nombre">Nombre:</label>
      <input type="text" id="nombre" name="nombre">
    </div>
    <div>
      <label for="email">Email:</label>
      <input type="email" id="email" name="email">
    </div>
  </fieldset>

  <fieldset>
    <legend>Preferencias de Notificación</legend>
    <div>
      <input type="checkbox" id="notif_email" name="notif_email">
      <label for="notif_email">Recibir notificaciones por email</label>
    </div>
    <div>
      <input type="checkbox" id="notif_sms" name="notif_sms">
      <label for="notif_sms">Recibir notificaciones por SMS</label>
    </div>
  </fieldset>

  <button type="submit">Guardar</button>
</form>

Reto

Crea un formulario para elegir una pizza. Usa un <fieldset> con la leyenda "Tamaño" para agrupar tres radio buttons ("Pequeña", "Mediana", "Grande"). Usa otro <fieldset> con la leyenda "Ingredientes Extra" para agrupar dos checkboxes ("Queso extra", "Piña").
