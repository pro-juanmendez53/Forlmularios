Lección 4: La Etiqueta <label>

Importancia de las Etiquetas

La etiqueta <label> es fundamental para la usabilidad y accesibilidad de los formularios. Proporciona una descripción textual para un elemento de entrada (un <input>, <textarea>, etc.).

Aunque podrías simplemente escribir texto al lado de un campo, usar <label> ofrece dos ventajas clave:

  1. Mejora la Usabilidad: Cuando un usuario hace clic en el texto de una etiqueta, el foco se mueve automáticamente al campo de entrada asociado. Esto es especialmente útil para elementos pequeños como checkboxes y radio buttons.
  2. Mejora la Accesibilidad: Los lectores de pantalla (software que usan las personas con discapacidad visual) utilizan las etiquetas para anunciar qué información se debe introducir en un campo de formulario. Sin etiquetas, el formulario sería muy difícil de usar para ellos.

Asociar Etiquetas con Entradas

Hay dos maneras de asociar una <label> con un campo de entrada. La más robusta y recomendada es usando los atributos for y id.

  1. El atributo id del campo de entrada (<input>) debe tener un valor único en la página.
  2. El atributo for de la etiqueta (<label>) debe coincidir exactamente con el valor del id del campo de entrada.

Ejemplo de Código

Observa cómo el atributo for de la etiqueta coincide con el id del input.


<form action="#">
  <div>
    <!-- Asociación explícita con 'for' e 'id' -->
    <label for="nombre_usuario">Nombre de usuario:</label>
    <input type="text" id="nombre_usuario" name="username">
  </div>

  <div>
    <input type="checkbox" id="recordarme" name="remember">
    <label for="recordarme">Recordarme en este equipo</label>
  </div>
</form>

Prueba a hacer clic en el texto "Nombre de usuario:" o "Recordarme en este equipo" en un formulario real. Verás que el cursor se activa en el campo correspondiente.

Reto

A continuación tienes un campo de texto y un checkbox sin etiquetas adecuadas. Modifica el código para asociar correctamente una <label> a cada uno de ellos usando el método for/id.
