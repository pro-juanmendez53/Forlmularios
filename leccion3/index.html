Lección 3: Elementos de Entrada (<input>)

La etiqueta <input>

La etiqueta <input> es, sin duda, el elemento más versátil y fundamental de los formularios HTML. Permite crear una gran variedad de campos de entrada gracias a su atributo type.

Tipos de <input> más comunes

  • type="text": Un campo de texto de una sola línea.
  • type="password": Un campo de texto donde los caracteres se ocultan.
  • type="email": Un campo para direcciones de correo electrónico (con validación básica).
  • type="number": Un campo para introducir números.
  • type="date": Un campo para seleccionar una fecha.
  • type="checkbox": Una casilla de verificación que puede estar marcada o no.
  • type="radio": Un botón de opción. Varios radios con el mismo name forman un grupo donde solo uno puede ser seleccionado.
  • type="submit": Un botón que envía el formulario.
  • type="reset": Un botón que resetea todos los campos del formulario a sus valores iniciales.
  • type="file": Un campo que permite al usuario seleccionar un archivo de su dispositivo.
  • type="hidden": Un campo de datos que no es visible para el usuario, pero cuyo valor se envía con el formulario.

Atributos Importantes

  • name: Esencial. Asigna un nombre al campo, que se usará como clave cuando se envíen los datos.
  • value: El valor inicial del campo. En el caso de los botones, es el texto que muestran.
  • placeholder: Un texto de ejemplo que aparece en el campo cuando está vacío.
  • required: Hace que el campo sea obligatorio.
  • disabled: Deshabilita el campo, el usuario no puede interactuar con él.
  • readonly: El usuario no puede modificar el valor del campo, pero sí se envía con el formulario.

Ejemplo de Código


<form action="#" method="GET">
  <div>
    <label for="nombre">Nombre:</label>
    <input type="text" id="nombre" name="usuario_nombre" placeholder="Escribe tu nombre">
  </div>
  <div>
    <label for="pass">Contraseña:</label>
    <input type="password" id="pass" name="usuario_pass" required>
  </div>
  <div>
    <input type="submit" value="Enviar">
  </div>
</form>

Reto

Crea un formulario que pida al usuario su email y su fecha de nacimiento. Añade un botón de envío. Asegúrate de usar los tipos de input correctos y de que el campo de email tenga un placeholder.
