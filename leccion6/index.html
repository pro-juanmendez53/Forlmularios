Lección 6: Listas Desplegables (<select>, <option>)

Crear Menús de Selección

Las listas desplegables son una excelente manera de permitir que el usuario elija una o varias opciones de una lista predefinida. Esto reduce errores de escritura y estandariza las respuestas. Se crean combinando dos etiquetas:

  • <select>: El contenedor del menú desplegable.
  • <option>: Cada uno de los ítems dentro de la lista.

El atributo name se coloca en la etiqueta <select>. El valor que se envía al servidor es el que se encuentra en el atributo value de la <option> seleccionada.

Opción Seleccionada por Defecto

Para que una opción aparezca seleccionada por defecto cuando se carga la página, se le añade el atributo booleano selected.

Grupos de Opciones con <optgroup>

Si tienes una lista muy larga, puedes agrupar opciones relacionadas bajo un título usando la etiqueta <optgroup>. El atributo label del <optgroup> define el título del grupo, que no es una opción seleccionable.

Ejemplo de Código


<form action="#">
  <div>
    <label for="pais">Elige tu país:</label>
    <select id="pais" name="user_country">
        <option value="">--Por favor, elige una opción--</option>
        <option value="gt">Guatemala</option>
        <option value="es" selected>España</option>
        <option value="ar">Argentina</option>
        <option value="co">Colombia</option>
    </select>
  </div>

  <div>
    <label for="vehiculo">Elige tu vehículo preferido:</label>
    <select id="vehiculo" name="user_vehicle">
        <optgroup label="Europeos">
            <option value="volvo">Volvo</option>
            <option value="audi">Audi</option>
        </optgroup>
        <optgroup label="Asiáticos">
            <option value="toyota">Toyota</option>
            <option value="honda">Honda</option>
        </optgroup>
    </select>
  </div>
</form>

Reto

Crea una lista desplegable para que un usuario elija su nivel de estudios. Las opciones deben ser: "Primaria", "Secundaria", "Bachillerato", "Universidad" y "Postgrado". La opción "Bachillerato" debe estar seleccionada por defecto.
