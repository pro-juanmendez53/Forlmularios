Lección 12: Ejemplo Completo de Formulario

Uniendo todas las piezas

En esta lección, vamos a construir un formulario de registro completo que utilice muchos de los elementos y conceptos que hemos aprendido a lo largo de la clase.

Este ejemplo práctico te ayudará a ver cómo interactúan las diferentes partes de un formulario para crear una experiencia de usuario coherente y funcional.

Formulario de Registro de Evento

A continuación se muestra el código completo de un formulario de registro para un evento ficticio. Analiza cómo se utilizan las etiquetas, los tipos de input, las validaciones y la agrupación de campos. 


<form action="/registrar-evento" method="POST">
  <fieldset>
    <legend>Información Personal</legend>
    <div>
      <label for="nombre">Nombre Completo:</label>
      <input type="text" id="nombre" name="nombre_completo" required placeholder="Ej: Ana García">
    </div>
    <div>
      <label for="email">Correo Electrónico:</label>
      <input type="email" id="email" name="email_usuario" required placeholder="tu@email.com">
    </div>
  </fieldset>

  <fieldset>
    <legend>Detalles del Evento</legend>
    <div>
      <label for="tipo_entrada">Tipo de Entrada:</label>
      <select id="tipo_entrada" name="tipo_entrada" required>
        <option value="">-- Selecciona una opción --</option>
        <option value="general">Entrada General</option>
        <option value="vip">Entrada VIP</option>
        <option value="estudiante">Entrada de Estudiante</option>
      </select>
    </div>
    <div>
      <p>¿Qué talleres te interesan?</p>
      <input type="checkbox" id="taller_html" name="intereses" value="html">
      <label for="taller_html">Taller de HTML Avanzado</label>
      <br>
      <input type="checkbox" id="taller_css" name="intereses" value="css">
      <label for="taller_css">Taller de CSS Grid</label>
      <br>
      <input type="checkbox" id="taller_js" name="intereses" value="js">
      <label for="taller_js">Taller de JavaScript Moderno</label>
    </div>
  </fieldset>

  <fieldset>
    <legend>Comentarios y Preferencias</legend>
    <div>
      <label for="comentarios">¿Alguna alergia o comentario adicional?</label>
      <textarea id="comentarios" name="comentarios" rows="4" placeholder="Déjanos un mensaje..."></textarea>
    </div>
    <div>
      <input type="checkbox" id="terminos" name="acepta_terminos" required>
      <label for="terminos">Acepto los términos y condiciones.</label>
    </div>
  </fieldset>

  <div class="nav-links">
    <button type="submit">Registrarme en el Evento</button>
    <button type="reset">Limpiar Formulario</button>
  </div>
</form>

Reto

¡Es tu turno de brillar! Crea tu propio formulario completo desde cero. El tema es libre. Puede ser un formulario de contacto, de registro para un videojuego, para pedir una pizza, para adoptar una mascota, etc.

Tu formulario debe incluir, como mínimo:

  • Al menos tres tipos diferentes de <input>.
  • Un <textarea>.
  • Un <select> con varias opciones.
  • Al menos un campo con validación (ej. required).
  • Al menos un <fieldset> para agrupar campos.
  • Botones de envío y reseteo.

¡Sé creativo y aplica todo lo que has aprendido!
