Lección 7: Botones

Tipos de Botones en Formularios

Los botones son el principal mecanismo de acción en un formulario. Permiten al usuario confirmar, cancelar o realizar otras acciones. Hay dos formas principales de crear botones: con la etiqueta <input> o con la etiqueta <button>.

Botones con <input>

Ya los hemos visto brevemente. Son simples y directos.

  • <input type="submit">: Su acción por defecto es enviar el formulario. El texto del botón se define con el atributo value.
  • <input type="reset">: Su acción por defecto es limpiar todos los campos del formulario, devolviéndolos a su estado inicial. El texto también se define con value.
  • <input type="button">: Un botón genérico sin acción por defecto. Es una "pizarra en blanco" que requiere JavaScript para ser útil.

La etiqueta <button>

La etiqueta <button> es más moderna y flexible. A diferencia de <input>, es una etiqueta de apertura y cierre. Esto permite incluir contenido más rico dentro del botón, como texto con formato (negritas, cursivas) e incluso imágenes.

El comportamiento del <button> se define con su atributo type:

  • <button type="submit">: (Valor por defecto si se omite) Envía el formulario.
  • <button type="reset">: Limpia el formulario.
  • <button type="button">: No hace nada por defecto, necesita JavaScript.

¡Cuidado! Si usas <button> dentro de un <form> y omites el type, actuará como un botón de envío (submit), lo que puede causar envíos de formulario no deseados.

Ejemplo de Código


<form action="#">
  <!-- Campos del formulario aquí -->
  <label for="user-text">Dato:</label>
  <input type="text" id="user-text">

  <h4>Botones con <input></h4>
  <input type="submit" value="Enviar (input)">
  <input type="reset" value="Limpiar (input)">

  <h4>Botones con <button></h4>
  <button type="submit">
    <strong>Enviar</strong> (button)
  </button>
  <button type="reset">Limpiar (button)</button>
  <button type="button" onclick="alert('Hola Mundo!')">
    Botón con JavaScript
  </button>
</form>

Reto

Crea un formulario con un campo de texto. Añade dos botones creados con la etiqueta <button>: uno de tipo "submit" que diga "Registrarse" y otro de tipo "reset" que diga "Empezar de nuevo".
